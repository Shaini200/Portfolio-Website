# Shaini Kavindya – Portfolio (Jordan Codes Style)

A clean, multi-page portfolio website inspired by the **Jordan Codes** Webflow template,
built with pure HTML, CSS, and JavaScript — no frameworks, no build tools.

## 📁 File Structure

```
jordan_portfolio/
├── index.html        ← Home page
├── about.html        ← About, Experience, Education, Skills
├── portfolio.html    ← All 6 projects (full detail)
├── contact.html      ← Contact form + info
├── css/
│   └── style.css     ← All styles (dark teal theme)
├── js/
│   └── main.js       ← Nav, animations, form logic
└── images/           ← Add your images here!
    ├── profile.png   ← Your profile photo (recommended: 500×500 or 600×800)
    ├── n1.png        ← Face Recognition project screenshot
    ├── n2.png        ← Chatting Application screenshot
    ├── n3.png        ← University Management screenshot
    ├── n4.png        ← Text to Voice screenshot
    ├── n5.png        ← Weather App screenshot
    └── n6.png        ← Screen Recorder screenshot
```

## 🚀 Getting Started

1. **Add your images** to the `images/` folder (see above)
2. **Open** `index.html` in any browser — no server needed!

## 🎨 Customisation

All colours are CSS variables in `css/style.css` under `:root`:

```css
--accent:  #0aafa5;   /* Main teal — change this to any colour */
--bg:      #0d0f0e;   /* Dark background */
--text:    #e8ede9;   /* Body text */
```

To update your CV link, search for `drive.google.com` in all HTML files and replace.
